Projet de démonstration/point de départ pour développer une application liée aux histoires de rue (site web, API ou script). Personnalise selon tes besoins.

## Essayer rapidement

1. Ouvre le dossier dans VS Code :

```bash
code /home/marin/Documents/histoire-dans-la-rue
```

2. Commence à développer dans `src/` : crée des fichiers `.py`, `.js`, ou `index.html` selon ton stack.

3. Si tu veux initialiser un projet Node.js :

```bash
cd /home/marin/Documents/histoire-dans-la-rue
npm init -y
```

4. Si tu veux un projet Python minimal :

```bash
cd /home/marin/Documents/histoire-dans-la-rue
python3 -m venv .venv
source .venv/bin/activate
```
